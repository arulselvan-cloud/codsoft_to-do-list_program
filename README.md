# 📝 To-Do List Manager in C++

A simple and interactive **console-based To-Do List Manager** developed using **C++**.  
This application allows users to manage daily tasks efficiently through a menu-driven interface.

---

# 📌 Project Overview

The To-Do List Manager helps users:

- Add new tasks
- View all tasks
- Mark tasks as completed
- Remove tasks from the list

This project demonstrates the use of:
- Structures
- Vectors
- Functions
- Loops
- Conditional Statements
- User Input/Output

---

# ⚙️ How the Program Works

## Step 1
The user selects an option from the menu.

```text
1. Add Task
2. View Tasks
3. Mark Task as Completed
4. Remove Task
5. Exit
```

## Step 2
The selected operation is performed.

## Step 3
The program continues running until the user chooses to exit.

---

# ▶️ How to Run the Program

## Compile the Program

```bash
g++ main.cpp -o todo
```

## Run the Program

```bash
./todo
```
